# AL NAJAYEB GROUP Website Landing Page

Static GitHub Pages landing page for **مجموعة النجايب للخدمات والاستشارات** / **AL NAJAYEB GROUP FOR SERVICES AND CONSULTING**.

## Included files

- `index.html` — redesigned bilingual under-construction homepage
- `styles.css` — responsive dark navy and gold corporate styling
- `assets/najayeb-brand.png` — uploaded brand image used in the homepage design

## Upload to GitHub

Upload all files and folders in this package to the root of your GitHub repository, then enable GitHub Pages for the branch that contains `index.html`.
